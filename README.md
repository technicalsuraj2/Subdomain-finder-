# Subdomain-finder-
Subfinder is a fast Python tool for discovering hidden subdomains of a target domain. Built for recon, bug bounty, and security research. Lightweight, simple, and effective. Developed by Suraj Patel.
## ⚡ Termux Installation

```bash
pkg update && pkg upgrade -y
pkg install git python -y
git clone https://github.com/technicalsuraj2/Subdomain-finder-.git
cd Subdomain-finder-
bash install.sh
```

## Usage

```bash
python subfinder.py example.com
```
